# Projeto 01 - Testes Funcionais em E-commerce (SauceDemo)

## Objetivo

Validar as principais funcionalidades de um sistema de e-commerce por meio de testes manuais, identificando falhas e documentando os resultados obtidos.

## Escopo

* Login de usuário
* Cadastro de usuário
* Listagem de produtos
* Carrinho de compras
* Checkout
* Finalização de compra

## Tipos de Teste Aplicados

* Testes Funcionais
* Testes Exploratórios
* Testes End-to-End (E2E)
* Validação de Campos

## Resultados

* 10 Casos de Teste Executados
* 6 Casos Aprovados
* 4 Casos Reprovados
* 4 Bugs Identificados

## Estrutura do Projeto

* casos-de-teste.md
* bug-reports.md
* evidencias/

## Ferramentas Utilizadas

* Google Chrome
* Microsoft Excel
