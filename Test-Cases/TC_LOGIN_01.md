# TC_LOGIN_01 - Login com credenciais válidas

## Pré-condição
Usuário previamente cadastrado.

## Passos

1. Acessar a página de login.
2. Inserir e-mail válido.
3. Inserir senha válida.
4. Clicar em "Entrar".

## Resultado Esperado

Usuário autenticado e redirecionado para a página inicial.

## Resultado Obtido

Login realizado com sucesso.

## Status

✅ Aprovado
